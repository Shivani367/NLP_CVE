Maximum Entropy Markov Model

Unlike HMMs which are purely generative, the Maximum Entropy Markov Model (MEMM) uses a discriminative Logistic Regression transition model. It combines the current observation ($o_t$) with the previous state ($s_{t-1}$) to predict the probability distribution of the next state.
